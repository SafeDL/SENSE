========================================================================================================================================================================================================
METRIC INTERPRETATION
========================================================================================================================================================================================================

1. Raw Failures: Number of test cases triggering collisions (unfiltered)
2. Representative Failures: Unique/deduplicated failure cases
3. AUC-FDC: Area Under Failure Discovery Curve
   - Larger values indicate better early-stage search efficiency
   - Reflects how quickly the algorithm discovers failures

4. nAUC-FDC: Normalized AUC = AUC / (T_total × N_max)
   - Normalizes AUC by total budget and discovered failures
   - Enables fair comparison across different experiments
   - Larger values indicate better efficiency per unit budget

5. Coverage Rate: Percentage of grid cells (from grid search) found by algorithm
6. Search Time: Total wall-clock time for the search process
7. Total Evals: Sum of surrogate calls and real simulations
8. Real Sims: Actual CARLA simulations executed
9. Surrogate: Fast predictions from surrogate model
10. Captured Grid Cells: Number of unique failure domain cells discovered
11. Ground Truth Cells: Total number of collision cells in the test space
12. Efficiency: Failures found per second of search time
